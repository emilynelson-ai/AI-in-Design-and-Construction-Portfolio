Development Methodology

How This Website Was Built

This website was developed through an iterative process of prompting, reviewing, testing, and refining code with ChatGPT.

This website was developed through an iterative coding process using ChatGPT to draft, revise, and refine the HTML structure for each page of the portfolio.

I began by defining the site goals, required pages, design preferences, and content priorities, then used follow-up prompts to replace generic placeholder language with course-specific and professionally relevant material.

As each page evolved, I tested the code in GitHub, identified issues with layout, file paths, embedded media, and image rendering, and then used targeted prompts to troubleshoot and improve those problems step by step.

This process also involved progressively aligning the homepage, about page, AI news archive, and AEC use cases page so the full site felt visually cohesive and clearly connected to the course outcomes.

Overall, the methodology combined human direction, content curation, and design judgment with AI-assisted coding support, showing how iterative prompting can function as a practical workflow for building and refining a complete website.
